# Semaine 15/16

## Critères d'évaluation du mini-projet

1. Qualité de la présentation
   1. Présentation dynamique et intéressante.
   2. Respect du temps à disposition 5-7 minutes.
   3. Démo fonctionnelle.
2. Readme
   1. Décrit le projet
   2. Explique comment utiliser le projet
   3. Explique comment installer et développer sur le projet
3. Code
   1. Le prof arrive à clone, poetry install
   2. Le prof arrive à run votre projet
   3. Le prof arrive à poetry run pytest qui démontre au moins un test unitaire.
